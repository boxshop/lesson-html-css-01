“HTML is a markup language that web browsers use to interpret and compose text, images, and other material into visual or audible web pages”

HTML is root of a website (or web application). It's basic to dive into web development.

Structure of HTML


<!DOCTYPE html>
<html>
  <head>
    <title>Title will be in Tab</title>
  </head>
  <body>
   <!-- This is a comment -->
    <p>Content display</p>
  </body>
</html>

Just say Hello World!
